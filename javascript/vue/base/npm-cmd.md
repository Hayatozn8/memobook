# 相关指令
- 手动配置webpack时，需要的loader
    ```sh
    npm i vue-loader -D
    npm i vue-style-loader -D
    npm i vue-template-compiler -D
    ```
- 安装 vue-cli
    ```sh
    npm install -g @vue/cli

    # 检查版本
    vue -V
    ```
- vue-cli2 相关指令
    - 拉取 vue-cli2 的模板
        ```sh
        npm install -g @vue/cli-init
        ```
    - vue-cli2 创建项目
        ```sh
        vue init webpack <project_name>
        ```
- vue-cli3 相关指令
    - vue-cli3 创建项目
        ```sh
        vue create <project_name>
        ```
