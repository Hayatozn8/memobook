# 网页开发的简介
[top](#catalog)
- 客户端的形式
    - 文字客户端
    - 图形化界面
        - 通过点击拖动等来使用软件
        - 包括:windows、macos、Andriod、ios中的大部分应用
        - 属于C/S架构
    - <label style="color:red">>网页</label>
        - 通过访问网页来使用软件，所有的网站都属于这个范畴
        - 属于B/S架构
- 网页的优点
    - 不需要安装
    - 无需更新
    - 跨平台，开发一次处处使用
- 网页中使用的语言
    - HTML
    - CSS
    - JavaScript
- **网页的编写需要遵守W3C规范**
- **网页的结构**
    - 根据W3C标准，一个网页主要有3部分组成：
        1. 结构：HTML用于描述页面的结构
        2. 表现：CSS用于控制页面中元素的样式
        3. 行为：JavaScript用于响应用户操作
